# NLP Coursework: Patronising and Condescending Language Detection

Binary classification model for SemEval-2022 Task 4 Subtask 1: detecting patronising and condescending language (PCL) towards vulnerable communities.

## Prerequisites

### 1. Install Python dependencies

```bash
pip install -r requirements.txt
```

### 2. Clone the dataset repository

```bash
git clone https://github.com/Perez-AlmendrosC/dontpatronizeme
```

### 3. Download the PCL data file

The `dontpatronizeme_pcl.tsv` file is not included in the repository and must be downloaded manually from the task organisers. Place it at:

```txt
dontpatronizeme/semeval-2022/dontpatronizeme_pcl.tsv
```
